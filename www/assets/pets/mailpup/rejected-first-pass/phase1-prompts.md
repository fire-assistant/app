# Phase 1 Generation Prompts

Shared visual lock for every prompt:

Small chibi firefighter puppy sprite, same identity as the source reference. Cream floppy ears, round peach cheeks, tiny black eyes, small mouth, navy firefighter cap with a small yellow badge, orange firefighter shirt with dark navy collar, tiny paws, compact rounded body. Thick dark 1-2 px outline, flat cel colors, pixel-art-adjacent digital pet style, simple readable silhouette. Center each frame in a 192 x 208 cell. No text, no UI, no speech bubbles, no scenery, no cast shadow, no floor shadow, no loose effects. Use a perfectly flat solid #00ff00 chroma-key background.

## surprised

Create a 6-frame horizontal sprite strip for a simple startled reaction.
Frame sequence:
1. Neutral standing pose.
2. Eyes widen, ears start to lift.
3. Body pops upward slightly, paws lift near chest.
4. Peak surprise, round open mouth, ears out.
5. Settling down, paws lowering.
6. Almost neutral but still alert.
