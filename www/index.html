    HOW TO USE

    6가지 도구, 이렇게 쓰세요

    각 기능을 탭하면 사용법과 주의사항을 확인할 수 있습니다.

    🧯
    01 소방시설 탐색기
    ›

    건물 용도·면적·층수 등 조건을 입력하면 현행 법령 기준으로 설치 의무가 있는 소방시설 목록과 설치 이유를 자동으로 도출합니다.

    1탐색기 진입 후 모드 선택 — 간단한 버전(빠른 도출) 또는 자세한 버전(건축허가일 기준)
    2용도·연면적·층수·스프링클러 여부 등 질문에 순서대로 답하기
    3결과 화면에서 설치 필요 시설 목록과 설치 이유 확인
    💡 자체점검 실시결과 보고서를 받을 때, 해당 특정소방대상물에 설치해야 할 소방시설의 종류들이 뭔지 참고할 수 있습니다.
    👥
    02 다중이용업소 탐색기
    ›

    업종을 선택하고 질문에 답하면 다중이용업소 해당 여부를 판정하고, 해당 시 설치해야 하는 안전시설 목록을 안내합니다.

    1업종 선택 — 음식점·노래연습장·PC방·고시원 등
    2면적·층수·영업 형태 등 조건 질문에 답하기
    3판정 결과와 설치해야 할 안전시설 목록 확인
    💡 동일 업종이라도 영업장 면적·층·복층 여부에 따라 결과가 달라질 수 있습니다.
    📅
    03 법정기한 계산기
    ›

    자체점검 제출기한, 안전관리자 선임기한, 부적합 조치기한을 자동으로 계산합니다.

    1상단 탭에서 계산 유형 선택 — 자체점검 / 안전관리자 / 부적합조치
    2기준일·점검일·선임해제일 등 필요한 날짜 입력
    3결과 확인 후 "메인화면에 표시"로 알림판에 저장
    💡 알림판에 저장하면 D-7 이내 기한이 빨간색으로 강조됩니다.
    📋
    04 자체점검 가이드
    ›

    자체점검 대상 여부 판정과 실시결과 보고서 읽기·작성 방법을 안내합니다.

    1상단 탭으로 페이지 이동 — 표지(1p)·대상물정보(2p)·소방시설선택(3p)·점검표(4~8p)
    2각 항목을 탭하면 해당 칸의 작성 방법과 읽는 요령 표시
    33p에서 등급 또는 시설을 선택하면 해당 점검표 항목만 필터링
    💡 보고서를 처음 받았을 때 어디를 봐야 할지 모를 때 유용합니다.
    📖
    05 소방시설 도감
    ›

    소방시설별 개요·종류·구성·설치기준을 한눈에 확인합니다. 탐색기 결과에서 모르는 시설이 나왔을 때 바로 찾아볼 수 있습니다.

    1목록에서 시설 선택 — 소화기·스프링클러·감지기·유도등 등
    2개요·종류·구성요소 확인
    3설치기준 탭에서 면적·층수별 설치 조건 확인
    💡 탐색기 결과에서 모르는 소방시설이 있으면 여기서 바로 찾아보세요.
    🧮
    06 유틸리티 도구함
    ›

    수용인원 산정, 소방안전관리보조자 선임 필요인원 계산 등 현장에서 바로 쓸 수 있는 소형 계산 도구 모음입니다.

    1계산 유형 선택 — 수용인원 / 보조자 선임인원
    2용도·면적·세대수 등 필요한 값 입력
    3산정 결과와 적용 기준 공식 확인

    앱처럼 설치하기

    📲
    INSTALL 바탕화면·홈 화면에 바로가기 추가

    메인화면 하단의 바로가기 추가 버튼을 누르면 기기에 맞는 방법이 안내됩니다.

    PC .url 파일을 다운로드해 바탕화면에 두고 더블클릭으로 실행
    Android Chrome 메뉴 → 홈 화면에 추가
    iPhone Safari 공유 버튼 → 홈 화면에 추가

    ✅ 설치 후엔 주소창 없이 전체화면으로 실행되며, 한 번 로딩 뒤에는 인터넷 없이도 기본 기능을 사용할 수 있습니다.
